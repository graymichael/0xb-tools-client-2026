## What 0xB Tools Includes

0xB Tools bundles a small set of practical browser-based helpers that work locally and do not rely on any backend service. In one lightweight interface, it offers Base64 encode/decode, text case conversion, a password strength checker, and a TDEE calorie calculator.

It is intended for users who want fast-loading web tools that operate entirely in the browser and do not require server-side processing. Alongside the utility tools, the project also ships with an interactive WebGL swirl and fluid-inspired demo, so it serves both everyday tasks and simple visual exploration.

## How to Use It

Pick the tool that matches the job you need done:

- Paste text into the Base64 tool to encode or decode it.
- Enter content into the case converter to switch capitalization styles.
- Fill in the fields in the TDEE calculator to estimate daily calorie needs.
- Type a password into the strength checker to review its score and feedback.
- Open the WebGL demo to view the animated swirl or fluid-style effect.

Since everything executes in the browser, the results stay local to the current session and no server round trip is needed.
